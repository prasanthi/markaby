markaby by prasanthi

Markaby extended to accommodate HTML 5.0 tags

Use the HTML 5.0 doctype to produce HTML 5.0 tags, header, footer, nav, article. This project could be modified by adding other HTML 5.0 tags and attributes as well. An example '.mab' file looks like: html_5 do header do h1 'Property Management System' p do a 'Home - ', :href => "/main/"; a 'Administrator', :href=> "/administrators/" end end nav do h1 'Navigation' ul do li { a 'Buy', :href=> '/sales/' } li { a 'Rent', :href=> '/rents/' } li { a 'Post Preferences', :href=> '/preferences/new' } end end footer do p 'Copyright Â© 2008 Prasanthi C Mahesh' p do a 'About_Us ', :href=> '/about_us/' end end end
